# Simple Mall
## 简介

**技术栈：Vue3 + Vuex**

**[在线预览](https://simple-mall.vercel.app/#/)**

使用vue3开发的一个简易的购物应用，购物车和地址模块使用vuex进行状态管理，主要完成前端部分：

- 注册、登录（默认不认证账号）
- 商场产品展示
- 购物车流程
- 订单模块
- 地址管理

## 安装

- 获取项目代码

```bash
git clone https://github.com/Blank6682/simple-mall.git
or
git clone git@github.com:Blank6682/simple-mall.git

```

- 安装依赖

```bash
cd mall-mode

yarn 
```

- 运行

```bash
yarn serve
```
